# Agro Forte, Futuro Sustentável 🌾🚀

Um website institucional e interativo focado no agronegócio moderno, destacando a união entre a alta produtividade no campo e a preservação ambiental através da tecnologia.

## 📋 Sobre o Projeto

O projeto **Agro Forte, Futuro Sustentável** foi desenvolvido com o objetivo de demonstrar como a inovação tecnológica (como drones, sensores e inteligência artificial) atua como pilar fundamental para uma agricultura eficiente, ecológica e virada para o futuro.

O site conta com uma interface responsiva, um design inspirado nas cores da natureza e uma secção interativa que apresenta factos reais sobre a tecnologia no campo.

## 🌟 Funcionalidades

- **Design Responsivo:** Adaptável a ecrãs de computadores, tablets e smartphones.
- **Navegação Suave (Smooth Scroll):** Menu que desliza suavemente até à secção selecionada.
- **Secção Interativa:** Um gerador de factos dinâmico em JavaScript que mostra inovações do agro sustentável ao clicar num botão.
- **Estética Moderna:** Paleta de cores cuidadosamente selecionada (tons de verde e terra) para transmitir a mensagem de sustentabilidade e tecnologia.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias web fundamentais:

* [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) - Estrutura do site.
* [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização, layout e responsividade.
* [JavaScript (ES6)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Lógica de interatividade e manipulação do DOM.
